## Signature Work

### EPC Tender Screening MCP

MCP-only, agent-agnostic tool server for evidence-based EPC tender screening.

- **Why it matters:** Turns tender documents and company profiles into reviewable bid/no-bid analysis without pretending the agent has data it does not have.
- **Built with:** TypeScript, MCP, HTTP transport, stdio transport, Zod, deterministic evals.
- **Proof points:** 10 MCP tools, `screen_tender` orchestration, evidence traces, OAuth-compatible connector routes, SSRF-conscious public source search.
- **Links:** [Showcase repository](https://github.com/Shiverion/epc-tender-screening-mcp-showcase)

### VerdictBench

Research benchmark comparing Long Context, Dense RAG, and Multi-Stage RAG on Indonesian Constitutional Court verdicts.

- **Why it matters:** Tests when RAG is still useful even as LLM context windows grow.
- **Built with:** Python, Gemini 2.5 Flash, GPT-4o Mini, FAISS, BM25, IndoBERT.
- **Proof points:** 50 verdicts, 300 human-reviewed QA pairs, Dense RAG statistically tied with Long Context while being 16-25x cheaper.
- **Links:** [Repository](https://github.com/Shiverion/VerdictBench-LCvsRAG) | [Paper](https://openreview.net/forum?id=46hzq45LPE)

### Distill / Paprika

Production AI paper-to-insights app that turns PDFs or academic paper URLs into interactive learning formats.

- **Why it matters:** Moves beyond flat summaries into structured outputs people can actually study and present.
- **Built with:** Next.js 15, React 19, TypeScript, Gemini 2.5 Flash, Firebase Auth, Firestore, Zod, Vercel.
- **Proof points:** 7 output types, native Gemini PDF handling, PDF or URL input, credit system, fast Firebase ID-token verification.
- **Links:** [Live demo](https://distill.shiverion.com) | **Source:** private

### InterviewMate AI

AI-powered virtual interviewer and candidate evaluation platform.

- **Why it matters:** Gives recruiters a real-time voice interview workflow with structured candidate assessment.
- **Built with:** TypeScript, Firebase, OpenAI Realtime API, WebRTC.
- **Proof points:** Real-time voice interviews, resume personalization, recruiter dashboard, automated candidate evaluation.
- **Links:** [Live demo](https://interviewmate-ai.shiverion.com/) | [Repository](https://github.com/Shiverion/interviewmate-ai)
